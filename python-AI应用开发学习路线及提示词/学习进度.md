### 我真正理解了什么

- `.env` 是本地开发中常用的配置方式，适合作为开发环境下的真实密钥存放位置，但不是生产环境最终密钥管理形态
- 企业开发中，代码本身不保存真实密钥，代码只负责通过 `os.getenv(...)` 或统一配置模块读取密钥
- `.gitignore` 的作用是防止敏感文件被误提交到 Git，但它不是完整的安全机制
- `OpenAI` SDK 本质是兼容 OpenAI 协议的 HTTP 客户端封装，不是模型本身
- 大模型多轮对话的“记忆”不是模型天然记住，而是程序通过维护 `messages` 列表实现上下文传递
- `System Prompt` 不只是“给 AI 一个角色”，更是在控制回答风格、边界和业务行为
- token 统计不仅是技术字段，也是企业项目中成本控制和监控的重要依据

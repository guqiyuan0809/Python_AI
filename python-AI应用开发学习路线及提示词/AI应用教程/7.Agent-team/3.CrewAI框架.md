# CrewAI 框架

## 什么是 CrewAI

CrewAI 是一个专门用于构建多 Agent 协作系统的 Python 框架。它的核心理念是模拟真实团队：每个 Agent 是一个"团队成员"，有自己的角色、目标和工具。

## 核心概念

### Agent（智能体）

一个 Agent 就是一个团队成员，定义了：
- **role**：角色名称（如"高级研究员"）
- **goal**：这个 Agent 的目标
- **backstory**：背景故事，帮助 AI 更好地进入角色
- **tools**：可以使用的工具列表

```python
from crewai import Agent

researcher = Agent(
    role="高级研究分析师",
    goal="发现关于 {topic} 的最新趋势和信息",
    backstory="你是一位经验丰富的研究分析师，擅长从海量信息中提取关键洞察。",
    tools=[search_tool],
    verbose=True
)
```

### Task（任务）

定义 Agent 需要完成的具体工作：
- **description**：任务描述
- **expected_output**：期望的输出格式
- **agent**：负责执行的 Agent

```python
from crewai import Task

research_task = Task(
    description="研究 {topic} 的最新发展趋势，收集至少 5 个关键信息点。",
    expected_output="一份结构化的研究报告，包含关键发现和数据支撑。",
    agent=researcher
)
```

### Crew（团队）

把 Agent 和 Task 组合成一个团队：
- **agents**：团队成员列表
- **tasks**：任务列表
- **process**：执行模式（sequential 串行 / hierarchical 层级）

```python
from crewai import Crew, Process

crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, writing_task],
    process=Process.sequential,  # 串行执行
    verbose=True
)

result = crew.kickoff(inputs={"topic": "AI Agent 发展趋势"})
```

## CrewAI 工作流程

```
1. 定义 Agent（角色、目标、工具）
2. 定义 Task（任务描述、期望输出、分配给谁）
3. 创建 Crew（组合 Agent 和 Task，选择执行模式）
4. 启动执行（crew.kickoff()）
5. 获取结果
```

## 安装

```bash
pip install crewai crewai-tools
```

## CrewAI 的优势

- 代码简洁，上手快
- 角色扮演机制让 Agent 输出质量更高
- 内置多种工具（搜索、文件读写等）
- 支持串行和层级两种模式
- 社区活跃，文档完善

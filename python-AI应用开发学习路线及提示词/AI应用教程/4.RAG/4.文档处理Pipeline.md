# 文档处理 Pipeline / paɪpˌlaɪn/

## 什么是文档处理 Pipeline？

文档处理 Pipeline 是将原始文档转换成可检索的向量数据的完整流程。

### 完整流程

```
原始文档（PDF/Word/Markdown/HTML）
  ↓
1. 文档加载（Document Loading）
  ↓
2. 文本提取（Text Extraction）
  ↓
3. 文本清洗（Text Cleaning）
  ↓
4. 文本分块（Text Chunking）★ 核心
  ↓
5. 向量化（Embedding）
  ↓
6. 存入向量数据库
```

这个 Pipeline 的质量直接决定了 RAG 系统的效果。

## 1. 文档加载（Document Loading）

### 支持的文档格式

| 格式 | 常用库 | 难度 |
|------|--------|------|
| TXT | 内置 `open()` | ⭐ |
| Markdown | `open()` | ⭐ |
| PDF | `PyPDF2`, `pdfplumber` | ⭐⭐ |
| Word | `python-docx` | ⭐⭐ |
| HTML | `BeautifulSoup` | ⭐⭐ |
| Excel | `pandas` | ⭐⭐ |
| 网页 | `requests` + `BeautifulSoup` | ⭐⭐⭐ |

### 实现：加载不同格式的文档

```python
import os
from typing import List, Dict

class DocumentLoader:
    """文档加载器"""
    
    @staticmethod
    def load_txt(file_path: str) -> str:
        """加载 TXT 文件"""
        with open(file_path, 'r', encoding='utf-8') as f:
            return f.read()
    
    @staticmethod
    def load_markdown(file_path: str) -> str:
        """加载 Markdown 文件"""
        with open(file_path, 'r', encoding='utf-8') as f:
            return f.read()
    
    @staticmethod
    def load_pdf(file_path: str) -> str:
        """加载 PDF 文件"""
        import PyPDF2
        
        text = ""
        with open(file_path, 'rb') as f:
            reader = PyPDF2.PdfReader(f)
            for page in reader.pages:
                text += page.extract_text() + "\n"
        return text
    
    @staticmethod
    def load_word(file_path: str) -> str:
        """加载 Word 文件"""
        from docx import Document
        
        doc = Document(file_path)
        text = "\n".join([para.text for para in doc.paragraphs])
        return text
    
    @staticmethod
    def load_html(file_path: str) -> str:
        """加载 HTML 文件"""
        from bs4 import BeautifulSoup
        
        with open(file_path, 'r', encoding='utf-8') as f:
            soup = BeautifulSoup(f.read(), 'html.parser')
            # 移除 script 和 style 标签
            for script in soup(["script", "style"]):
                script.decompose()
            return soup.get_text()
    
    @staticmethod
    def load_directory(dir_path: str) -> List[Dict]:
        """加载目录下的所有文档"""
        documents = []
        
        for root, dirs, files in os.walk(dir_path):
            for file in files:
                file_path = os.path.join(root, file)
                ext = os.path.splitext(file)[1].lower()
                
                try:
                    if ext == '.txt':
                        text = DocumentLoader.load_txt(file_path)
                    elif ext == '.md':
                        text = DocumentLoader.load_markdown(file_path)
                    elif ext == '.pdf':
                        text = DocumentLoader.load_pdf(file_path)
                    elif ext in ['.doc', '.docx']:
                        text = DocumentLoader.load_word(file_path)
                    elif ext in ['.html', '.htm']:
                        text = DocumentLoader.load_html(file_path)
                    else:
                        continue
                    
                    documents.append({
                        'text': text,
                        'metadata': {
                            'source': file_path,
                            'filename': file,
                            'extension': ext
                        }
                    })
                except Exception as e:
                    print(f"加载 {file_path} 失败: {e}")
        
        return documents

# 使用示例
loader = DocumentLoader()

# 加载单个文件
text = loader.load_pdf("document.pdf")

# 加载整个目录
documents = loader.load_directory("./docs")
print(f"加载了 {len(documents)} 个文档")
```

## 2. 文本提取与清洗

### 常见问题

1. **多余空格和换行**
2. **特殊字符和乱码**
3. **HTML 标签残留**
4. **页眉页脚**
5. **表格和图片说明**

### 实现：文本清洗

```python
import re

class TextCleaner:
    """文本清洗器"""
    
    @staticmethod
    def clean(text: str) -> str:
        """清洗文本"""
        # 1. 移除多余空格
        text = re.sub(r'\s+', ' ', text)
        
        # 2. 移除特殊字符（保留中英文、数字、常用标点）
        # text = re.sub(r'[^\w\s\u4e00-\u9fff.,!?;:()（）。，！？；：]', '', text)
        
        # 3. 移除 HTML 标签残留
        text = re.sub(r'<[^>]+>', '', text)
        
        # 4. 移除多余换行
        text = re.sub(r'\n+', '\n', text)
        
        # 5. 去除首尾空格
        text = text.strip()
        
        return text
    
    @staticmethod
    def remove_headers_footers(text: str, patterns: List[str]) -> str:
        """移除页眉页脚"""
        for pattern in patterns:
            text = re.sub(pattern, '', text)
        return text
    
    @staticmethod
    def normalize_whitespace(text: str) -> str:
        """规范化空白字符"""
        # 将所有空白字符替换为单个空格
        text = ' '.join(text.split())
        return text

# 使用示例
cleaner = TextCleaner()

raw_text = """
    这是一段   有很多    空格的文本。
    
    
    还有很多换行。
    <div>HTML 标签</div>
"""

clean_text = cleaner.clean(raw_text)
print(clean_text)
# 输出: "这是一段 有很多 空格的文本。 还有很多换行。 HTML 标签"
```

## 3. 文本分块（Text Chunking）★ 核心

文本分块是 RAG Pipeline 中最关键的一步，直接影响检索质量。

### 为什么要分块？

1. **AI 上下文窗口有限**：不能把整本书都塞进去
2. **提高检索精度**：小块更容易匹配用户问题
3. **降低成本**：只检索相关的小块，不是整个文档

### 分块策略对比

| 策略 | 说明 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| 固定长度 | 每 N 个字符切一段 | 简单 | 可能切断语义 | 通用 |
| 按段落 | 保持段落完整 | 语义完整 | 长度不均 | 结构化文档 |
| 按句子 | 保持句子完整 | 语义完整 | 块太小 | 短文本 |
| 递归分块 | 先按章节，再按段落 | 层次清晰 | 实现复杂 | 长文档 |
| 重叠分块 | 相邻块有重叠 | 避免信息断裂 | 存储冗余 | 推荐使用 |
| 语义分块 | 按语义边界切分 | 最优 | 计算成本高 | 高质量要求 |

### 推荐策略：重叠分块

```python
class TextChunker:
    """文本分块器"""
    
    @staticmethod
    def chunk_by_length(text: str, chunk_size: int = 500, overlap: int = 50) -> List[str]:
        """
        按固定长度分块（带重叠）
        
        Args:
            text: 原始文本
            chunk_size: 每块的字符数
            overlap: 重叠字符数
        
        Returns:
            分块后的文本列表
        """
        chunks = []
        start = 0
        
        while start < len(text):
            end = start + chunk_size
            chunk = text[start:end]
            chunks.append(chunk)
            start += chunk_size - overlap  # 重叠
        
        return chunks
    
    @staticmethod
    def chunk_by_paragraph(text: str, max_chunk_size: int = 1000) -> List[str]:
        """
        按段落分块
        
        Args:
            text: 原始文本
            max_chunk_size: 单个块的最大字符数
        
        Returns:
            分块后的文本列表
        """
        paragraphs = text.split('\n\n')
        chunks = []
        current_chunk = ""
        
        for para in paragraphs:
            if len(current_chunk) + len(para) <= max_chunk_size:
                current_chunk += para + "\n\n"
            else:
                if current_chunk:
                    chunks.append(current_chunk.strip())
                current_chunk = para + "\n\n"
        
        if current_chunk:
            chunks.append(current_chunk.strip())
        
        return chunks
    
    @staticmethod
    def chunk_by_sentence(text: str, sentences_per_chunk: int = 5) -> List[str]:
        """
        按句子分块
        
        Args:
            text: 原始文本
            sentences_per_chunk: 每块包含的句子数
        
        Returns:
            分块后的文本列表
        """
        import re
        
        # 简单的句子分割（中英文）
        sentences = re.split(r'[。！？.!?]\s*', text)
        sentences = [s.strip() for s in sentences if s.strip()]
        
        chunks = []
        for i in range(0, len(sentences), sentences_per_chunk):
            chunk = '。'.join(sentences[i:i+sentences_per_chunk])
            chunks.append(chunk)
        
        return chunks

# 使用示例
chunker = TextChunker()

text = "这是第一段。" * 100 + "\n\n" + "这是第二段。" * 100

# 固定长度分块（推荐）
chunks = chunker.chunk_by_length(text, chunk_size=500, overlap=50)
print(f"分成了 {len(chunks)} 个块")

# 按段落分块
chunks = chunker.chunk_by_paragraph(text, max_chunk_size=1000)
print(f"分成了 {len(chunks)} 个块")
```

### 分块参数选择指南

| 文档类型 | chunk_size | overlap | 说明 |
|---------|-----------|---------|------|
| 短文本（FAQ）| 200-300 | 20-30 | 保持问答完整 |
| 中等文档（文章）| 500-800 | 50-100 | 平衡语义和检索 |
| 长文档（书籍）| 1000-1500 | 100-200 | 保留更多上下文 |
| 代码文件 | 300-500 | 50 | 保持函数完整 |
| 对话记录 | 200-400 | 50 | 保持对话连贯 |

**经验法则**：
- chunk_size：根据文档类型和 AI 模型的上下文窗口调整
- overlap：通常是 chunk_size 的 10-20%
- 先用默认值（500/50），根据效果调整

### 实现：完整的文档处理器

```python
from typing import List, Dict
import hashlib

class DocumentProcessor:
    """完整的文档处理器"""
    
    def __init__(self, chunk_size: int = 500, overlap: int = 50):
        self.loader = DocumentLoader()
        self.cleaner = TextCleaner()
        self.chunker = TextChunker()
        self.chunk_size = chunk_size
        self.overlap = overlap
    
    def process_file(self, file_path: str) -> List[Dict]:
        """
        处理单个文件
        
        Returns:
            List[Dict]: 每个 dict 包含 text 和 metadata
        """
        # 1. 加载文档
        ext = os.path.splitext(file_path)[1].lower()
        if ext == '.pdf':
            text = self.loader.load_pdf(file_path)
        elif ext == '.txt':
            text = self.loader.load_txt(file_path)
        elif ext == '.md':
            text = self.loader.load_markdown(file_path)
        else:
            raise ValueError(f"不支持的文件格式: {ext}")
        
        # 2. 清洗文本
        text = self.cleaner.clean(text)
        
        # 3. 分块
        chunks = self.chunker.chunk_by_length(
            text, 
            chunk_size=self.chunk_size, 
            overlap=self.overlap
        )
        
        # 4. 添加元数据
        processed_chunks = []
        for i, chunk in enumerate(chunks):
            chunk_id = self._generate_chunk_id(file_path, i)
            
            processed_chunks.append({
                'text': chunk,
                'metadata': {
                    'source': file_path,
                    'filename': os.path.basename(file_path),
                    'chunk_id': chunk_id,
                    'chunk_index': i,
                    'total_chunks': len(chunks),
                    'chunk_size': len(chunk)
                }
            })
        
        return processed_chunks
    
    def process_directory(self, dir_path: str) -> List[Dict]:
        """处理整个目录"""
        all_chunks = []
        
        # 加载所有文档
        documents = self.loader.load_directory(dir_path)
        
        for doc in documents:
            # 清洗
            text = self.cleaner.clean(doc['text'])
            
            # 分块
            chunks = self.chunker.chunk_by_length(
                text,
                chunk_size=self.chunk_size,
                overlap=self.overlap
            )
            
            # 添加元数据
            for i, chunk in enumerate(chunks):
                chunk_id = self._generate_chunk_id(doc['metadata']['source'], i)
                
                all_chunks.append({
                    'text': chunk,
                    'metadata': {
                        **doc['metadata'],  # 继承原有元数据
                        'chunk_id': chunk_id,
                        'chunk_index': i,
                        'total_chunks': len(chunks)
                    }
                })
        
        return all_chunks
    
    @staticmethod
    def _generate_chunk_id(file_path: str, chunk_index: int) -> str:
        """生成唯一的 chunk ID"""
        file_hash = hashlib.md5(file_path.encode()).hexdigest()[:8]
        return f"{file_hash}_chunk_{chunk_index}"

# 使用示例
processor = DocumentProcessor(chunk_size=500, overlap=50)

# 处理单个文件
chunks = processor.process_file("document.pdf")
print(f"处理后得到 {len(chunks)} 个文本块")

# 处理整个目录
chunks = processor.process_directory("./docs")
print(f"处理后得到 {len(chunks)} 个文本块")

# 查看第一个块
print(chunks[0])
```

## 5. 完整的 Pipeline 示例

把所有步骤整合起来：

```python
import chromadb
from openai import OpenAI

class RAGPipeline:
    """完整的 RAG Pipeline"""
    
    def __init__(self, db_path="./chroma_db", collection_name="documents"):
        # 文档处理器
        self.processor = DocumentProcessor(chunk_size=500, overlap=50)
        
        # 向量数据库
        self.client = chromadb.PersistentClient(path=db_path)
        self.collection = self.client.get_or_create_collection(
            name=collection_name,
            metadata={"hnsw:space": "cosine"}
        )
        
        # OpenAI 客户端
        self.openai_client = OpenAI()
    
    def ingest_directory(self, dir_path: str):
        """
        摄入整个目录的文档
        
        流程：加载 → 清洗 → 分块 → 向量化 → 存储
        """
        print(f"开始处理目录: {dir_path}")
        
        # 1. 处理文档
        chunks = self.processor.process_directory(dir_path)
        print(f"处理完成，共 {len(chunks)} 个文本块")
        
        # 2. 批量向量化和存储
        batch_size = 100
        for i in range(0, len(chunks), batch_size):
            batch = chunks[i:i+batch_size]
            
            # 提取文本和元数据
            texts = [chunk['text'] for chunk in batch]
            metadatas = [chunk['metadata'] for chunk in batch]
            ids = [chunk['metadata']['chunk_id'] for chunk in batch]
            
            # 存入向量数据库（Chroma 会自动向量化）
            self.collection.add(
                documents=texts,
                metadatas=metadatas,
                ids=ids
            )
            
            print(f"已处理 {min(i+batch_size, len(chunks))}/{len(chunks)}")
        
        print("摄入完成！")
    
    def search(self, query: str, top_k: int = 3):
        """搜索相关文档"""
        results = self.collection.query(
            query_texts=[query],
            n_results=top_k
        )
        
        return {
            'documents': results['documents'][0],
            'metadatas': results['metadatas'][0],
            'distances': results['distances'][0]
        }
    
    def get_stats(self):
        """获取统计信息"""
        count = self.collection.count()
        return {
            'total_chunks': count,
            'collection_name': self.collection.name
        }

# 使用示例
if __name__ == "__main__":
    # 初始化 Pipeline
    pipeline = RAGPipeline()
    
    # 摄入文档
    pipeline.ingest_directory("./docs")
    
    # 查看统计
    stats = pipeline.get_stats()
    print(f"数据库中共有 {stats['total_chunks']} 个文本块")
    
    # 搜索
    results = pipeline.search("如何学习 Python？", top_k=3)
    for doc, metadata, distance in zip(
        results['documents'], 
        results['metadatas'], 
        results['distances']
    ):
        print(f"\n距离: {distance:.3f}")
        print(f"来源: {metadata['source']}")
        print(f"内容: {doc[:100]}...")
```

### 技巧 2：保留文档结构

```python
def chunk_with_structure(text: str) -> List[Dict]:
    """保留章节结构的分块"""
    chapters = text.split('\n# ')  # 按一级标题分割
    
    chunks = []
    for chapter in chapters:
        lines = chapter.split('\n')
        chapter_title = lines[0]
        chapter_content = '\n'.join(lines[1:])
        
        # 对每个章节分块
        chapter_chunks = chunk_by_length(chapter_content, 500, 50)
        
        for i, chunk in enumerate(chapter_chunks):
            chunks.append({
                'text': chunk,
                'metadata': {
                    'chapter': chapter_title,
                    'chunk_in_chapter': i
                }
            })
    
    return chunks
```

## 7. 常见问题

### Q1：chunk_size 应该设置多大？

**没有标准答案**，取决于：
- 文档类型（短文本 vs 长文档）
- AI 模型的上下文窗口
- 检索精度要求

**建议**：从 500 开始，根据效果调整。

### Q2：overlap 有必要吗？

**非常有必要**。没有 overlap 可能导致：
- 关键信息被切断
- 跨块的语义丢失

**建议**：overlap 设置为 chunk_size 的 10-20%。

### Q3：如何处理表格和图片？

**表格**：
- 转换成 Markdown 格式
- 或者提取表格内容作为独立块

**图片**：
- 提取图片说明文字
- 或者使用多模态模型（GPT-4V）提取图片内容

### Q4：如何处理代码文件？

```python
from langchain.text_splitter import Language, RecursiveCharacterTextSplitter

# 专门的代码分块器
splitter = RecursiveCharacterTextSplitter.from_language(
    language=Language.PYTHON,
    chunk_size=500,
    chunk_overlap=50
)

chunks = splitter.split_text(code)
```

## 总结

文档处理 Pipeline 是 RAG 的基础，关键要点：

1. **加载**：支持多种格式（PDF、Word、Markdown）
2. **清洗**：移除噪音，规范化文本
3. **分块**：重叠分块（chunk_size=500, overlap=50）
4. **元数据**：记录来源、位置等信息
5. **存储**：批量存入向量数据库

**推荐配置**：
- chunk_size: 500
- overlap: 50
- 分块策略：RecursiveCharacterTextSplitter

掌握文档处理 Pipeline 后，就能构建高质量的 RAG 系统了。
